# Make That Pipe Sound

For German version see below.


# Idea

Make simple pipe instruments and play them together.
Everyone gets a single pipe of a defined length and has to hit on one end of the pipe.
Each pipe has a color to distinguish it.
[This web app](https://kopp.github.io/make-that-pipe-sound/) shows the colors to hit on.
This makes a quick and simple orchestra, quite some fun and even some team building.


## Instructions: Build The Pipes

Pick a PVC Pipe with 25mm diameter and about 1mm wall thickness, e.g. "EN25 320N".
They are typically used to build electric wiring in your house and can be bought cheaply from your local DIY warehouse.
Use the following table to cut the pipes into segments of the given length.
Then use sandpaper to make the ends smooth.
Finally add duct tape in the respective color to them to ensure that it is easy to identify the pipe.

You may use apps such as
[Nail the Pitch](https://play.google.com/store/apps/details?id=com.autumnrockdev.nailthepitch&utm_source=emea_Med)
to verify that the pipe lengths produce the correct sounds for you.

Note: make your pipe 1.0595 times longer to get a pitch one half-note deeper (and 1.122 longer for a full note).
Apply this to the values given below to add more notes.

| pitch     | color              | length [mm] |
|-----------|--------------------|-------------|
| G3        | red red            | 417         |
| C4        | yellow             | 317         |
| D4        | green              | 283         |
| E4        | blue               | 253         |
| F4        | white              | 233         |
| G4        | red                | 210         |
| A4        | black              | 185         |
| A4#       | yellow and black   | 178         |
| B4        | brown              | 161         |
| C5        | yellow yellow      | 153         |



# Deutsch

Baue einfache Rohrinstrumente und spielt sie zusammen.
Jede Person erhält ein einzelnes Rohr einer festgelegten Länge und muss an einem Ende des Rohrs anschlagen.
Jedes Rohr hat eine Farbe zur Unterscheidung.
[Diese Web-App](https://kopp.github.io/make-that-pipe-sound/) zeigt die zu spielenden Farben an.
Das ergibt ein schnelles und einfaches Orchester, macht viel Spaß und fördert sogar Teambuilding.


## Anleitung: Rohre bauen

Wähle ein PVC-Rohr mit 25 mm Durchmesser und etwa 1 mm Wandstärke, z. B. „EN25 320N“.
Sie werden typischerweise für Elektroinstallationen im Haus verwendet und können günstig in Baumärkten erworben werden.
Verwende die folgende Tabelle, um die Rohre auf die angegebenen Längen zu schneiden.
Schleife danach die Enden mit Schleifpapier glatt.
Klebe zum Schluss Klebeband in der jeweiligen Farbe an, damit die Rohre leicht zu identifizieren sind.

Du kannst Apps wie bspw.
[Nail the Pitch](https://play.google.com/store/apps/details?id=com.autumnrockdev.nailthepitch&utm_source=emea_Med)
verwenden, um zu überprüfen, ob die Rohrlängen für dich die richtigen Töne ergeben.

Hinweis: Mache dein Rohr 1,0595-mal länger, um einen Halbton tiefer zu erhalten (und 1,122-mal länger für einen ganzen Ton).
Wende dies auf die unten angegebenen Werte an, um weitere Töne zu erzeugen.

| Ton       | Farbe              | Länge [mm] |
|-----------|--------------------|------------:|
| G3        | rot rot            | 417         |
| C4        | gelb               | 317         |
| D4        | grün               | 283         |
| E4        | blau               | 253         |
| F4        | weiß               | 233         |
| G4        | rot                | 210         |
| A4        | schwarz            | 185         |
| A4#       | gelb und schwarz   | 178         |
| B4        | braun              | 161         |
| C5        | gelb gelb          | 153         |
